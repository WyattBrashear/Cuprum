# Cuprum I
## A console for dogs and their humans.
### Overview
The Cuprum I is a console that is able to be played by a dog and their human. It features a custom controller in the
shape of a dog toy.
### Game
The game is Squeaky Clicker. It is a clicker game focus around squeaky dog toys. You can employ dogs for the "Autoclick"
functionality. And you can upgrade your squeaky for more points per squeak.
### Controller
The controller is in the shape of a dog bone with FSRs to detect how hard you squeeze it.
